2. 进入开发者后台，找到自建应用：

3. 左侧【应用功能】→【网页】，开启网页能力

   - FEISHU_APP_SECRET = "el81wzxrWcWvuVML5qrPGeOqNA0fp4R5"
4. 自动创建SQLite用户数据表：
```sql
CREATE TABLE IF NOT EXISTS users (
    user_id TEXT PRIMARY KEY,
    name TEXT,
```

5. 实现接口：

6. 所有原有业务接口统一接入该鉴权中间件，必须登录后才可访问

## 五、整体业务流程
1. 前端点击飞书登录→跳转飞书授权页
2. 授权通过自动回调前端callback地址，获取code
3. code传给后端，后端调用飞书API获取飞书用户唯一ID、姓名、头像、邮箱
4. 后端生成随机登录令牌，存入SQLite关联飞书用户
5. 前端保存令牌，后续所有请求自动携带
6. 后端所有接口强制鉴权，完成登录态控制
7. 支持拉取飞书企业通讯录，用于系统人员关联、权限分配
